## Git Add

#### Verifica quais áreas os arquivos estão alocados:
```
git status
```

#### Adiciona um arquivo ao stage:

```
git add file.txt
```

#### Adiciona todos os arquivos ao stage:
```
git add .
```
#### Remover um arquivo do stage
```
git rm --cached README.md
```

#### Remover um arquivo do stage com reset
```
git reset README.md
```
